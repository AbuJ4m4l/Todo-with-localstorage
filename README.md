A simple Todo application that stores tasks in the browser's localStorage. This project provides CRUD (Create, Read, Update, Delete) operations using localStorage to manage tasks effectively.

## Features

- Create tasks
- Read tasks
- Update tasks
- Delete tasks

## Installation

To run the Todo Application on your local machine, follow these steps:

### Prerequisites

- Ensure you have Node.js installed on your machine. If not, download and install it from [nodejs.org](https://nodejs.org/).

### Setup

1. **Clone the repository:**

```bash
git clone https://github.com/AbuJ4m4l/Todo-with-localstorage.git
```

2. **Navigate to the project directory:**

```bash
cd Todo-with-localstorage
```

3. **Install dependencies:**

```bash
npm install
```

### Running the project

1. **Build the project:**

```bash
npm run build
```

2. **Start the development server:**

```bash
npm run dev
```

After starting the development server, visit `http://localhost:3000` in your browser to use the Todo Application.
